## commands

```bash
# Compiler
rustc --version
rustc main.rs

# Cargo
cargo version
cargo new <project-name>
cargo check
cargo build
cargo build --release
cargo run

## Documentation based on project's dependencies 
cargo doc --open

# Toolchain
rustup docs --book
rustup doc
rustup update

# Bash
./target/debug/tsu-rust
```
